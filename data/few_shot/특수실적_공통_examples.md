# 특수실적_공통(T2) 추출 예시

`특수실적_공통` 필드는 공동수급체 전 구성원·모든 입찰자에게 **공통 적용되는 실적 기준** 원문.
구성원 별로 다른 선택적 요건이 아니라, 반드시 모두가 충족해야 하는 공통 조건만 해당된다.

## 주의

> **잘못 추출하기 쉬운 패턴 (negative 1 참고):**
> - 선택적 실적(택1) 또는 대표사에만 적용되는 조건을 `특수실적_공통`으로 오인하는 경우.
> - 특수실적 요건이 단 하나만 있을 때 그것을 무조건 공통으로 분류하는 오류.

---

## Positive 1 — 표준 원문 추출

공고문 원문:
```
공통실적 요건:
모든 구성원은 최근 3년간 도로공사 준공 실적이 5억 원 이상이어야 한다.
```

출력:
```json
{
  "특수실적_공통": "모든 구성원은 최근 3년간 도로공사 준공 실적이 5억 원 이상이어야 한다."
}
```

---

## Positive 2 — 동의어·형식 차이

공고문 원문:
```
특수실적 공통 기준
① 전 구성사: 건설업 면허 등록 후 5년 이상 경과
② 전 구성사: 공공기관 발주 토목공사 준공 실적 1건 이상
```

출력:
```json
{
  "특수실적_공통": "전 구성사: 건설업 면허 등록 후 5년 이상 경과 / 전 구성사: 공공기관 발주 토목공사 준공 실적 1건 이상"
}
```

규칙: 복수 항목이면 줄 구분 또는 " / "로 이어 붙여 원문 취지를 유지.

---

## Negative 1 — 잘못 추출하기 쉬운 패턴

공고문 원문:
```
대표사 요건: 최근 5년간 준공 실적 합계 50억 원 이상
구성사 요건: 최근 3년간 준공 실적 합계 10억 원 이상 (택1 중 하나 충족)
```

**잘못된 출력** (공통으로 오인):
```json
{
  "특수실적_공통": "최근 5년간 준공 실적 합계 50억 원 이상"
}
```

**올바른 출력:**
```json
{
  "특수실적_공통": null
}
```

이유: 대표사 전용 조건은 공통 적용 기준이 아니다. 구성사 선택적 조건은 `특수실적_내용` 해당.
