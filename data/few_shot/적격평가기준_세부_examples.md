# 적격평가기준_세부(T2) 추출 예시

`적격평가기준_세부` 필드는 적격심사 세부 평가 기준 텍스트 **원문 발췌**.
심사 항목별 배점·기준표·최저점수 등을 포함한다. 없으면 null.

## 주의

> **잘못 추출하기 쉬운 패턴 (negative 1 참고):**
> - 참가자격 조건(면허·실적 요건)을 적격평가기준으로 오인하는 경우.
> - 단순히 "적격심사"라는 단어가 나왔다고 해당 단락 전체를 추출하는 오류.
> - 기준 금액(평가기준금액 필드 대상)을 이 필드에 중복 삽입하는 오류.

---

## Positive 1 — 표준 원문 추출

공고문 원문:
```
적격심사 기준:
  - 시공능력 평가: 30점
  - 경영상태 평가: 30점
  - 신용평가: 20점
  - 하도급관리 계획: 20점
  최저점수: 60점 이상
```

출력:
```json
{
  "적격평가기준_세부": "시공능력 평가: 30점 / 경영상태 평가: 30점 / 신용평가: 20점 / 하도급관리 계획: 20점 / 최저점수: 60점 이상"
}
```

---

## Positive 2 — 동의어·형식 차이

공고문 원문:
```
낙찰자 결정 방법: 적격심사제 적용
배점: ① 기술능력 40 ② 가격 30 ③ 사회적책임 30
합산 95점 이상인 자 중 가격 최저자 낙찰
```

출력:
```json
{
  "적격평가기준_세부": "배점: ① 기술능력 40 ② 가격 30 ③ 사회적책임 30 / 합산 95점 이상인 자 중 가격 최저자 낙찰"
}
```

---

## Negative 1 — 잘못 추출하기 쉬운 패턴

공고문 원문:
```
참가자격:
  가. 면허 보유자
  나. 적격심사 통과 업체

낙찰: 최저가낙찰제 (적격심사 미적용)
```

**잘못된 출력** (적격심사 단어 오인):
```json
{
  "적격평가기준_세부": "적격심사 통과 업체"
}
```

**올바른 출력:**
```json
{
  "적격평가기준_세부": null
}
```

이유: 이 공고는 최저가낙찰제 적용으로 적격심사 세부 기준 자체가 없다. "적격심사"라는 단어만으로 이 필드를 채우면 안 된다.
