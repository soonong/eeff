```

├── _template/                ← 신규 작업 시 이 폴더를 통째로 복사
```

`{source_prefix}_{notice_id}`

| prefix | 출처 |
|--------|------|
| `g2b_` | 나라장터(G2B) |

## 2. 1건당 작업 흐름

```
[1] source 받음
      └─ source.html / source.pdf / source.txt 파일 추가

[2] 개인정보 마스킹 (redact)
      python -m scripts.redact --in data/ground_truth/{폴더}/source.* --inplace

[3] AI 초안 생성 (label_assist — D2 구현 예정)
      python -m scripts.label_assist data/ground_truth/{폴더}/

[4] diff 검수
      expected.json AI 초안 vs 정답을 키별로 확인/수정

[5] meta.yaml 완성
      reviewed_at: 오늘 날짜 (예: 2026-04-30)
```

## 3. expected.json 작성 가이드

### 기본 형식

```json
{
  "extracted": {
    "종목": [["포장공사업", "토목공사업"]],
    "투찰율": 0.87745,
    ...
  },
  "source": {
    "종목": "입찰참가자격: 지반조성·포장공사업 및 토목공사업",
    "투찰율": "낙찰하한율: 87.745%",
    ...
  }
}
```

- `extracted`: 정규화된 값 (타입 규칙 엄수)
- `source`: 원문에서 해당 값의 근거 발췌 (1~2문장, 원문 그대로)
- **공고에 없는 항목**: `null` 입력 (빈 문자열 `""` 사용 금지)

---

### 타입별 작성 예시

#### `nested_list[str]` — 종목

AND(및·겸유)는 같은 리스트 묶음, OR(또는·하거나)는 별도 묶음.

```json
"종목": [["포장공사업", "토목공사업"]]
// OR가 있으면:
"종목": [["포장공사업"], ["토목공사업"]]
```

#### `float` — 투찰율, 지역의무비율 등

0~1 사이 소수점 5자리.

```json
"투찰율": 0.87745
"지역의무비율": 0.49
```

#### `int` — 기초금액, 추정가격 등

원 단위 정수. 쉼표 없이.

```json
"기초금액": 3922300000
"추정가격": 3565727272
```

#### `str` (ISO 날짜시간) — 현설일, 투찰마감일 등

`yyyy-MM-ddTHH:mm` 형식.

```json
"투찰마감일": "2026-05-10T14:00"
"입력일": "2026-04-25"
```

#### `list[str]` — 지역

시도/시군구 이름 리스트.

```json
"지역": ["서울특별시"]
"지역": ["경기도 수원시", "경기도 성남시"]
```

#### `dict` — 보험료합계, 금액정보

키-값 쌍. 키는 원문 항목명 그대로.

```json
"보험료합계": {
  "국민연금": 51461963,
  "국민건강": 38948581,
  "산업안전": 46807852
}
```

#### `bool` — 종목_모두보유, 기초금액발표전 등

```json
"종목_모두보유": true
"기초금액발표전": false
```

#### `int` (0 또는 1) — 상호진출여부

```json
"상호진출여부": 1
```

#### `str` (enum) — 입찰방식, 일반_기타_전문, 공고상태

허용값만 입력:
- `입찰방식`: `"일반경쟁"` | `"제한경쟁"` | `"협상에 의한 계약"` | `"수의계약"`
- `일반_기타_전문`: `"일반"` | `"기타"` | `"전문"`
- `공고상태`: `"공고중"` | `"마감"` | `"취소"` | `"정정공고"`

#### `str` (자유 텍스트) — 특수실적, 적격평가기준_세부 등

원문 발췌 그대로. 줄바꿈 포함 가능.

```json
"특수실적": "최근 10년 이내 도로포장 100억원 이상 공사 실적 보유"
```

## 4. meta.yaml 필드 의미

```yaml
category: "g2b"         # g2b | hanjeon_dogang_lh | jichache | pdf_text | pdf_scan | edge_case
difficulty: 2           # 1(단순 표형식) | 2(복합 표+텍스트) | 3(스캔/엣지케이스)
labeler: "Lee"          # 라벨러 이름 또는 이니셜
reviewed_at: 2026-04-30 # ISO 날짜 (CEO 최종 검수 완료일)
notes: "정정공고 포함"   # 특이사항 (없으면 빈 문자열)
```

### category 기준

| 값 | 설명 |
|----|------|
| `g2b` | 나라장터 표준 HTML |
| `hanjeon_dogang_lh` | 한전·도공·LH 자체 사이트 |
| `jichache` | 지자체(시·구) |
| `pdf_text` | PDF 첨부 텍스트형 |
| `pdf_scan` | PDF 스캔본 (OCR) |
| `edge_case` | 정정공고·취소·협상계약 등 |

### difficulty 기준

| 값 | 설명 |
|----|------|
| 1 | 단순 표 형식, 모든 항목 명확히 기재 |
| 2 | 복합 표+단락, 항목 추론 필요 |
| 3 | OCR 노이즈, 비표준 서식, 정정/취소 등 엣지케이스 |

## 5. 자주 하는 실수

- `null` 대신 `""` 쓰는 것 — **금지**. null만 사용
- 금액에 쉼표 포함: `3,922,300,000` → **금지**. `3922300000`
- 날짜 형식 혼용: `2026.05.10` → **금지**. `2026-05-10T14:00`
- source에 원문 아닌 요약 쓰기 → **금지**. 원문 그대로 복사
- 종목 AND/OR 구분 오류 — 공고의 "및", "겸유" vs "또는", "하거나" 확인
